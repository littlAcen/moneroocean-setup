HOW IT WORKS
====================================================================

1. Checks if running as root (required for /etc/shadow)
2. Gets hostname or IP for file naming
3. Creates temp directory: /tmp/.exfil_RANDOM/
4. Copies /etc/passwd → HOSTNAME_passwd
5. Copies /etc/shadow → HOSTNAME_shadow
6. Calls send_email_with_attachments() function
7. Python3 sends email via SMTP with proper MIME attachments
8. Cleans up temp files
9. Done!

TROUBLESHOOTING
====================================================================

Q: No email received?
A: Check:
   1. RECIPIENT_EMAIL is set correctly (line 17)
   2. Script ran as root: id -u (should be 0)
   3. Python3 available: python3 --version
   4. Internet works: ping smtp.mailersend.net
   5. Port 587 not blocked: nc -zv smtp.mailersend.net 587
   6. Check spam folder
