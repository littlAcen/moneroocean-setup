  - Auto-start with improvements

====================================================================
TROUBLESHOOTING
====================================================================

Q: Script killed a legitimate process!
A: Edit is_our_miner() function to whitelist it

Q: Zen3 MSR values cause crashes?
A: Rare, but if it happens:
   wrmsr -a 0xc0011022 0xc000000401500000  # Old value
   wrmsr -a 0xc001102b 0x2000cc14          # Old value

Q: Kinsing killer using too much CPU?
A: Increase sleep from 60 to 300 seconds in the script

Q: Want to see what was killed?
A: Check /var/log/kinsing_killer.log (if enabled)

Q: How to verify miner improvements?
A: Check hashrate:
   journalctl -u swapd -n 100 | grep "speed"
